# template-fsharp-azure-fn

> Azure Function F# Template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao fsharp-azure-fn my-project
```

### From git

```bash
sao rand0me/template-fsharp-azure-fn my-project
```
